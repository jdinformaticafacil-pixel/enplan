⬇️ Exportar backup completo
Descarga una copia completa de todas las colecciones de Firestore en formato JSON. Incluye lugares, reseñas y artículos con todos sus campos.
📦 Colecciones incluidas: places · reviews · articles
📅 Formato: JSON con timestamps convertidos a ISO 8601
💾 Nombre del archivo: enplan-backup-YYYY-MM-DD.json
📋 Instrucciones de restauración
Para restaurar un backup, usa el script Python de importación con el archivo JSON generado, o importa manualmente desde Firebase Console usando la herramienta de importación de datos.
1. Guarda el archivo JSON en una ubicación segura
2. Se recomienda hacer backup semanal automático
3. Conserva al menos los últimos 4 backups
4. Para restaurar contacta al desarrollador o usa Firebase Console
